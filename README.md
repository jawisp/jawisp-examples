# Jawisp Examples

Jawisp 1.x examples

## How to Run

To build and run examples you need:

* Java 21 or later

Run examples:

- **`./gradlew :examples:quickstarts:run`**: Run quickstarts example

Test examples:

- **`./gradlew :examples:quickstarts:test`**: Test quickstarts example

## Documentation

Each example has a README that contains additional details for building and running the example.
